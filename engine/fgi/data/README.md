# #6 信用評価損益率（買い方）の確定日次履歴

`margin_pl_history.csv` は、松井証券「店内信用評価損益率の推移」（信用買い評価損益率）の
**確定日次データ（正のデータ）** です。date,value（% 単位）で、2026-02-02 以降の営業日を収録。

- 目的：#6 の履歴を MTM 近似（tier-2）だけに頼らず、**実データの水準で較正/上書き**する。
- 位置づけ：**tier-1（正）と同格の観測値**。tier-3 の較正オフセット算出と観測日の上書きに使う。
- 優先度：同一日付では **松井ライブ取得（`web/public/data/series/margin_pl_ratio.csv`）が上書き**。
  通常はライブと履歴は一致する（どちらも松井の確定値）。
- 更新方法：新しい期間を足す/値を直す場合はこの CSV を編集（date,value）。列名は `date,value`。

出典：松井証券「店内信用評価損益率の推移」。自作指標の較正/表示のための日次値。
